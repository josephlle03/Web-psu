Ejercicios de Bucles

Identificar patrones y repeticiones

  • Ejercicio 1

    • ¿Cuántas veces se repite el número 3 en la secuencia 3, 6, 9, ...?

Sumar Números Naturales

Pedir al usuario un número entero positivo y calcular la suma de todos los números naturales hasta ese número usando un bucle

  • Ejercicio 1

    • Si el usuario ingresa 5, el programa debería calcular 1+2+3+4+5 = 15.

Factorial de un Número

Pedir al usuario un número entero positivo y calcular su factorial usando un bucle.

  • Ejercicio 1

    • ¡Si el usuario ingresa 4, el programa debería calcular 4! = 432*1 = 24

Números Pares e Impares en un Rango

Pedir al usuario dos números enteros y mostrar todos los números pares e impares en ese rango usando bucles

  • Ejercicio 1

    • Si el usuario ingresa 3 y 9, el programa debería mostrar pares: 4, 6, 8 e impares: 3, 5, 7, 9.

Tabla de Multiplicar y/o Sumar

Pedir al usuario un número entero y tipo d tabla y generar su tabla de multiplicar o sumar del 1 al 10 usando un bucle.

  • Ejercicio 1

    • Si el usuario ingresa 7, y tabla “*”el programa debería mostrar 7x1=7, 7x2=14, ..., 7x10=70.

Secuencia de Fibonacci

Pedir al usuario un número entero positivo y generar esa cantidad de números de la secuencia de Fibonacci usando un bucle.

  • Ejercicio 1

    • Si el usuario ingresa 6, el programa debería mostrar 0, 1, 1, 2, 3, 5.
